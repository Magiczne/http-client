# http-client
[![npm][badge]][0]

Very simple, no dependency [`window.fetch`](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API) wrapper.

## Installation

```
npm install -g @magiczne/http-client
```

```
yarn global add @magiczne/http-client
```
